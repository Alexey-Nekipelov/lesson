# Домашнее задание:

**Возьмите любой CSV-файл (например, про погоду, автомобили, фильмы). Загрузите его в Pandas, выполните**:

* Выведите первые 5 строк и статистику числовых столбцов.

* Отфильтруйте строки по условию на ваш выбор.

* Постройте два графика: один линейный (по времени), один scatter или гистограмму.
Пришлите код и получившиеся графики (можно скриншоты)».
